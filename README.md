# Requirements

-   Node.js v19 or higher (Using Node v22.11 as of now)

# Installation

```
// to install
npm install

// to build
npm run build

// start server
npm run start:server

// start electron server (electron-forge)
npm run start

// start server on development
npm run dev
```

# Quick Install & Build then Run

-   `npm run setup`
-   `npm run start:server`
    #### `http://localhost:3000`
